Constellation Splash Screen - Design Review

Three visual alternatives to fix UI/UX issues

Current State

Current Constellation Design

Identified Issues:

  • Constellation Stars Too Large/Blob-like - Star glows create large blob effects that obscure the precise constellation patterns, making them look more like clouds than celestial bodies
  • Poor Text Contrast - Light text on dark indigo gradient lacks sufficient contrast, especially "Written in the Stars" tagline which is difficult to read
  • Generic Typography Hierarchy - Title treatment "بالعربي poetry" doesn't have enough visual weight or mystical quality to anchor the design
  • Overcomplicated Button Design - CTA button has too many competing effects (border, backdrop blur, hover glow) without clear visual priority
  • Theme Toggle Placement - Top-right button feels disconnected from the celestial theme and floats awkwardly

Option 1: Refined Constellation - Sharper Stars

Open Preview →

Design Choice:

Fixes the current direction by making stars sharp, precise points instead of blobs. Maintains the constellation concept but executes it with crisp, geometric precision inspired by astronomical charts.

Key Changes:

  • Sharp Star Points - 3-4px precise circles with tight glow (8-15px radius), creating clean constellation patterns
  • Enhanced Text Contrast - Upgraded to #f5f5f7 for tagline with text-shadow, purple gradient on Arabic title for emphasis
  • Cleaner Typography - Increased logo-ar to 5rem with text-shadow glow, better visual hierarchy
  • Simplified CTA - Single border, subtle backdrop blur, removed competing effects for clarity
  • Geometric Lines - Sharp 1px constellation lines with gradient fade at edges, more chart-like
  • Integrated Theme Toggle - Matches star aesthetic with purple tones and blur

Option 2: Minimal Cosmic - Pure Typography Focus

Open Preview →

Design Choice:

Completely different approach - removes interactive constellations entirely and focuses on massive, dramatic typography with subtle atmospheric starfield. Pure black background with minimal star patterns.

Key Changes:

  • Atmospheric Starfield - CSS radial-gradient star patterns instead of interactive SVG, ultra-minimal and performant
  • Massive Typography - Logo scaled to 7-8rem, becomes the hero element (previously 4-5rem)
  • Single Constellation Accent - One 160px line with star endpoints above logo, minimal visual anchor
  • Pure Text CTA - Replaces button with underlined text link, arrow on hover, ultra-clean
  • Pure Black Background - #08090f instead of gradient, enhances contrast and focus
  • No Theme Toggle - Embraces dark-only aesthetic for consistency
  • Reduced Description - Removes body copy, lets typography breathe

Option 3: Animated Cosmos - Dynamic Constellation Drawing

Open Preview →

Design Choice:

Radically different - constellations draw themselves on page load using SVG stroke-dashoffset animation. Stars appear with scale animation, lines connect with stroke drawing. Creates "being written in real-time" effect.

Key Changes:

  • SVG Line Drawing Animation - Constellation lines animate with stroke-dashoffset from 10 to 0 over 1s, sequential timing
  • Star Scale Animation - Stars scale from 0 to full size with opacity fade-in, synchronized with line drawing
  • Word-by-Word Tagline - "Written in the Stars" appears word-by-word with staggered delays (1.5s, 1.7s, 1.9s, 2.1s)
  • Orbital Icon Rings - Pen icon has rotating orbital rings (20s, 30s rotation), space-themed focus
  • Shimmer CTA Border - Gradient border animates with borderShimmer keyframe, premium feel
  • Aurora Background - Animated radial gradient that shifts vertically (auroraShift 15s), adds depth
  • Gradient Logo Text - Arabic text uses linear-gradient with background-clip for luxury effect

Recommendations

Best Overall: Option 1 (Refined) - Fixes all core issues while maintaining the interactive constellation concept. Sharp stars create clear patterns, improved contrast makes text readable, simplified CTA removes visual noise.

Best for Performance: Option 2 (Minimal) - CSS-only starfield with no JavaScript, fastest load time, pure typography focus creates dramatic impact without complexity.

Best for Engagement: Option 3 (Animated) - Constellation drawing animation creates "wow" moment on page load, word-by-word tagline builds anticipation, most memorable first impression.

Best for Accessibility: Option 1 or 2 - Both have excellent contrast ratios, Option 2 removes motion for reduced-motion preference, Option 1 maintains interactivity.

Implementation Recommendation: Start with Option 1 for immediate quality improvement. Consider Option 3 for marketing/landing page where animation impact is valued. Option 2 works best for ultra-fast loading on slow connections.
