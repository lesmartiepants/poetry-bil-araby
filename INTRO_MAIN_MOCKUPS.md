# Intro + Main App Mockups (10 Directions)

This set follows the UX/UI agent guidance: show, don't tell; visual options; distinct directions.

## Design Prompt (with 21st.dev component cues)

**Prompt:**
Design 10 distinct intro + main app mockups for an Arabic poetry experience. Each direction must pair a **landing intro** with a **main reading** screen. Keep typography-first layouts, balanced density, and cross‑platform clarity (desktop + mobile). Use 21st.dev component ideas as starting points: **hero split**, **bento grid**, **floating bottom bar**, **timeline/stepper**, **glass card**, **side navigation**, **chip filters**, **audio player**, **mosaic grid**, and **story scroll**. Ensure Arabic text is legible (Amiri or equivalent), and controls are finger-friendly with clear hierarchy.

## Intended Flow (ASCII)

```
[Intro / Landing]
   |
   v
[Pick poem / CTA] --> [Main Reading]
   |                      |
   v                      v
[Audio + Insight]     [Save / Share / Next]
```

## 10 Distinct Directions

1. **Celestial Lens** — cosmic orbits, lens‑style poem focus, floating controls.
2. **Calligraphic Minimal** — parchment, ink strokes, large calligraphy + quiet controls.
3. **Bento Atlas** — bento grid intro, modular cards, atlas side‑pane reading.
4. **Desert Horizon** — warm sunrise gradients, stamps, airy poem card.
5. **Ink Mono** — monochrome editorial, dot grid, typewriter‑style controls.
6. **Glass Pavilion** — frosted panels, layered glass cards, calm hierarchy.
7. **Library Catalog** — bookshelf nav, catalog index, archival reader pane.
8. **Rhythm Wave** — audio‑first waveform, rhythm controls, player focus.
9. **Mosaic Tiles** — geometric tile navigation, pattern framing for poems.
10. **Scroll Story** — chapter timeline, story arcs, sticky navigation controls.
