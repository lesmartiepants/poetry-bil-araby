Controls Redesign - UX Design Review

Three visual alternatives to fix control panel usability issues

Current State

1. Collapsed Sidebar (Default)

2. Expanded Sidebar

Identified Issues:

  • Hidden functionality - Text settings buried in nested Settings submenu (requires 2 clicks)
  • Expansion behavior - Sidebar expands/collapses, disrupting muscle memory and spatial stability
  • Inconsistent toggle states - Romanize uses visual border/opacity, but no clear binary indicator
  • Wrong order - Current order: Copy, Share, Romanize, Divider, Settings, Divider, Account (not matching requirements)
  • Dividers present - Visual separators break the clean, unified aesthetic
  • Day/night buried - Theme toggle nested in Settings submenu, should be prominent at top-right

Option 1: Minimalist Glass Panels

Open Preview →

Design Choice:

Same general direction as current design, but with clean glass morphism, no expansion, and logical layout. Keeps the aesthetic while fixing all usability problems.

Key Changes:

  • Always-visible sidebar - Fixed width (48px), no expansion, icons + labels always visible
  • Text settings panel (top-left) - Dedicated glass panel with T+/T− in circular buttons, clear toggle switches with sliding thumbs, font preview with Arabic glyph
  • Day/night toggle (top-right) - Glass pill with icon + label, clear state indication
  • No dividers - Unified visual language, separation achieved through spacing and subtle borders
  • Order: Save, Share, Copy, Ratchet, Sign In - Exactly as requested
  • Toggle clarity - ON state = gold background + border + subtle glow

Best for:

Teams who want to keep the general feel of the current design but fix all the UX problems in a straightforward, low-risk way.

Option 2: Dense Utility Layout

Open Preview →

Design Choice:

Completely different direction - everything in a single toolbar below the header, modeled after professional design tools. Maximizes discoverability and reduces visual footprint.

Key Changes:

  • Single horizontal toolbar - All text controls + theme toggle in one bar at the top
  • Toggle switches - Horizontal on/off switches with clear thumb position (very familiar to users)
  • Wider action buttons - 56px square buttons with icons + labels, densely packed
  • Vertical sidebar moves - Still on the right, but with larger touch targets and no expansion
  • Visual grouping - Thin vertical dividers separate text controls from theme toggle
  • Always visible - No hidden menus, everything at a glance

Best for:

Teams prioritizing efficiency and maximum discoverability. Users familiar with design tool interfaces will feel at home.

Option 3: Typographic Elegance

Open Preview →

Design Choice:

A radical departure focusing on minimalism, calligraphic detail, and tooltip-based discovery. Emphasizes the poetic aesthetic with ultra-slim UI.

Key Changes:

  • Edge bar instead of panels - Ultra-thin (60px) fixed top edge bar, feels like part of the frame
  • Circular buttons with tooltips - Text controls use small circular buttons with hints on hover; labels hidden to reduce noise
  • Elegant toggles - Minimalist toggle with bare-knob design when off, filled when on
  • Ultra-minimal sidebar - 44px icons only, no labels, tooltips appear on hover (gold on dark)
  • Artistic theme toggle - Circular emblem design with sun/moon icon in container
  • Typographic hierarchy - Uses Cormorant Garamond serif for body, creates sophisticated French literary feel

Best for:

Teams wanting a bold, design-forward statement that matches the mystical/poetic nature of the app. Highest aesthetic impact, moderate learning curve.

Recommendations

Best Overall: Option 1 (Minimalist Glass) - Solves all the problems with minimal change to visual language, maintains clarity, and offers the best balance of discoverability and elegance. The glass aesthetic fits the mystical vibe while being extremely clear.

Best for Mobile/Touch: Option 2 (Dense Utility) - Larger touch targets (56px) and clear labels make it the most mobile-friendly. The toolbar approach also works well on smaller screens where edge placement is tricky.

Best for Aesthetic Impact: Option 3 (Typographic Elegance) - If you want the UI to feel like part of the artwork, this is the winner. The edge bar concept and tooltip-based discovery create a unique experience. However, it sacrifices some discoverability and may need onboarding for first-time users.
