Scroll Reveal Poetry Designs

Three approaches to scroll-driven Arabic poetry experiences, inspired by 45r.jp/ja/indigo-hakusho/. Click "Open Full" to experience each design.

V1 — Horizontal Scroll

Vertical scroll input converts to horizontal content movement (RTL). Poems reveal as you traverse a wide horizontal strip, closest to the 45r.jp approach. Each poem section has its own color atmosphere with parallax decoration layers.

V2 — Depth Layers + Clip-Path

Vertical scroll with z-axis depth reveals. Each poem is hidden behind an expanding geometric mask (8-pointed star, Islamic arch, circle) that grows as you scroll. Five parallax depth layers create a "descending through a manuscript" feeling.

Clip-path Reveals 5-Layer Z-Depth Geometric Masks IntersectionObserver Color Transitions CSS Particles
Open Full Download

V3 — Three.js 3D WebGL

Full 3D experience using Three.js. Camera flies along a spline path through geometric chambers, each containing a poem rendered on floating text planes. Wireframe geometry, particle systems, and depth fog create a cinematic journey through a virtual manuscript world.
