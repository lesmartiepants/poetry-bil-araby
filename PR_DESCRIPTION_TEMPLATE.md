## Design Details

All mockups feature:
- **Arabic Typography:** Amiri font for poems with proper RTL layout
- **Dual Screens:** Intro/landing (left) + Main reading (right)
- **Mobile-Friendly:** Touch-optimized controls and ergonomic placement
- **Interactive HTML:** Each design has a corresponding HTML file in `mockups/`

See [`INTRO_MAIN_MOCKUPS.md`](INTRO_MAIN_MOCKUPS.md) for the complete design prompt and technical details.
