# Walkthrough Designs

This directory will contain walkthrough/onboarding flow designs for each feature theme.

## Status: Pending

Walkthrough designs need to be created for each splash theme to complete the splash -> walkthrough -> main app flow.

## Planned Walkthroughs

Each walkthrough should guide users through:
1. Welcome / Brand introduction
2. Navigate through poems
3. Listen to poetry (audio features)
4. Get deep insights (AI features)

## Design Guidelines

- Match the visual language of the corresponding splash design
- Use consistent typography from the splash variant
- 4-step guided tour with bilingual content (Arabic primary, English secondary)
- Step indicators, Previous/Next navigation, Skip option
- Glass morphism overlays matching theme aesthetics
- Mobile-first responsive design
