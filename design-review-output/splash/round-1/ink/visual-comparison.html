Technical Implementation:

  • SVG Text Element: <text direction="rtl">ÿßŸÑÿ¥ÿπÿ±</text> renders Arabic properly as a single string
  • Animated Gradients: Two <linearGradient> definitions with <animate> elements moving gradient stops over 3 seconds

  • SVG Filter: feGaussianBlur creates soft glow around active portion

Why This Approach Works:

  • Arabic ligatures maintained (ÿ± + ÿπ + ÿ¥ + ŸÑ + ÿß rendered as connected calligraphy)
  • Gradient animation creates smooth continuous reveal (not choppy individual letters)
  • RTL gradient direction (x1="100%", x2="0%") honors Arabic reading flow
  • Gold glow follows the reveal elegantly, mimicking ink being applied by a brush
  • Pure declarative animation (no JavaScript state management)
