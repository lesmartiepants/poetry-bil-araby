### How to Use

1. Open `design-review/review.html` in a browser
2. Use the **Feature** dropdown to select a design category
3. Use the **Component** dropdown to switch between Splash / Walkthrough / Main App / Controls
4. Use the **Option** dropdown to cycle through variants
5. Keyboard: `Left/Right` = features, `Up/Down` = options, `1-4` = component types, `I` = info panel, `F` = fullscreen
