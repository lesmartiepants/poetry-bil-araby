Light & Shadow Splash Screen - Design Review

Three visual alternatives to fix shadow quality, depth effects, and performance issues

Identified Issues:

  • Shadow Quality Too Muddy: Multiple overlapping shadow layers (base gradient + lattice overlay + ambient occlusion) create visual mud instead of organic depth
  • Light Rays Over-engineered: 5 separate rays + dappled pools compete for attention, animation feels mechanical
  • Lattice Pattern Static: Fixed SVG octagonal grid doesn't respond to light position, feels like Photoshop filter
  • Performance Issues: Multiple blur filters and 8+ values updating per frame cause rendering jank
  • Design Incoherence: Text shadows don't follow light source, button glow is static, no correlation between "light" and actual shadows

Option 1: Improved Chiaroscuro

Open Preview →

Design Choice:

Keep dramatic lighting concept, simplify execution. Same direction as current design but properly executed.

Key Changes:

  • Single unified shadow layer - Remove ambient occlusion and base layer, use one radial gradient that follows light position
  • Dynamic lattice mask - CSS repeating-linear-gradient that rotates with light angle (not static SVG)
  • 3 hero light rays (not 5) - Better contrast, clearer hierarchy, center ray is brightest
  • Text shadows follow light mathematically - shadowX/Y calculated from light position using CSS variables
  • Performance upgrade - CSS gradients instead of blur filters where possible, 60fps smooth

Technical Approach:

  • CSS custom properties (--light-x, --light-angle) drive all dynamic elements
  • Single requestAnimationFrame loop updates 4 variables (not 8+)
  • No SVG masks in critical render path

Option 2: Soft Depth with Layered Shadows

Open Preview →

Design Choice:

Completely different direction: subtle stratification instead of dramatic contrast. Inspired by Japanese shoji screens + morning fog.

Key Changes:

  • 3 distinct shadow layers - Background (deepest), Midground (transitional fog), Foreground (gentle veil)
  • No lattice pattern - Pure gradient stratification, no geometric overlay
  • Gentle light source - Radial glow that "lifts" shadows progressively, no rays
  • Frosted glass aesthetic - Depth through opacity shifts and subtle texture
  • Slower animation - 12-second breathing cycle (not 8) for more meditative feel

Visual Philosophy:

  • Depth emerges from layer separation, not light vs dark
  • Minimal movement - light breathes gently (±5% position change)
  • Softer, more contemplative aesthetic
  • Button uses frosted glass with backdrop-blur

Option 3: Minimal Ray Tracing

Open Preview →

Design Choice:

Radically different: ONE perfect light ray through architectural opening. Inspired by Pantheon oculus + Gothic cathedral windows.

Key Changes:

  • ONE dominant light ray - Vertical/diagonal volumetric beam, no competing elements
  • High contrast - Pure black (#000) background vs brilliant highlight
  • No pattern overlay - Pure volumetric lighting simulation, no lattice
  • Text emerges from darkness - Content brightness follows light intensity via CSS filter
  • Ultra-simple DOM - 3 elements total (darkness, ray, content), maximum performance

Technical Excellence:

  • Minimal DOM - fastest render path possible
  • Single light ray with ::before pseudo-element for edge definition
  • Content brightness controlled by one CSS variable (--brightness)
  • Dust particles in beam for realism (5 elements, CSS animation only)
  • Text literally invisible in shadow, brilliant in light

Recommendations

Best Overall: Option 1 (Improved Chiaroscuro) - Keeps the dramatic vision but fixes all technical issues. Proper shadow quality, dynamic lattice, coherent lighting. Same philosophy, better execution.

Best for Performance: Option 3 (Minimal Ray Tracing) - Ultra-simple DOM (3 elements), maximum FPS, works perfectly on low-end mobile devices. Most elegant code.

Best for Meditation/Poetry Context: Option 2 (Soft Depth Layers) - Contemplative aesthetic, slower breathing animation, gentle emergence from shadow. Most poetic interpretation of "light and shadow."

Most Dramatic Impact: Option 3 (Minimal Ray Tracing) - High contrast, single perfect beam, content literally emerges from complete darkness. Most cinematic.
