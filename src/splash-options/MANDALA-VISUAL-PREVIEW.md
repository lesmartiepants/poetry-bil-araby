## Responsive Behavior

### Mobile (320px - 640px)
- Mandala patterns: 90vw (fluid)
- Text: 14px base, clamp() for scaling
- Button: Full width with padding
- Walkthrough: Single column, 90% width
- Icon patterns: Simplified (fewer lines)

### Tablet (641px - 1024px)
- Mandala patterns: 500px fixed
- Text: 16px base
- Button: Auto width with generous padding
- Walkthrough: 80% width, max 600px
- Icon patterns: Full complexity

### Desktop (1025px+)
- Mandala patterns: 600px fixed
- Text: 18px base
- Button: Auto width with padding
- Walkthrough: Fixed 700px width
- Icon patterns: Full complexity with subtle glow
